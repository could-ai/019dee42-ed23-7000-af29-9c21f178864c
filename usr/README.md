# Corporate Presentation App

A robust Flutter application designed to showcase a professional corporate presentation. This app demonstrates how to build an interactive, slide-based interface completely in Flutter.

## Features Included
- **Interactive Slide Navigation**: Swipe gestures and button controls to navigate through the presentation.
- **Responsive Layout**: Adapts gracefully to mobile and desktop screen sizes.
- **Corporate Topics Addressed**:
  - Company Overview
  - Management System
  - Machine Specifications
  - Raw Material
  - Production Process
  - ETP (Effluent Treatment Plant)
  - Merchandising
  - Washing R&D Unit
- **Media Placeholders**: Built-in visual slots representing embedded PDFs and images for each slide.
- **Modern UI/UX**: Polished design using Material 3 with a professional corporate color scheme.

## Tech Stack
- Flutter
- Dart
- Material 3 Design System

## How to Run
1. Ensure you have Flutter installed and configured.
2. Clone this repository.
3. Run `flutter pub get` to fetch any dependencies.
4. Run `flutter run` to launch the app on your preferred connected device or emulator.
